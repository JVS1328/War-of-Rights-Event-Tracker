# Map Images

Place the following map images in this directory:

- `antietam.png` - Antietam map
- `finishedferry.png` - Harper's Ferry map
- `completemountain.png` - South Mountain map
- `drillcamp.png` - Drill Camp map

These maps will be used by the rangefinder tool to measure distances.

**Scale:** 1 pixel = 1 yard in-game
